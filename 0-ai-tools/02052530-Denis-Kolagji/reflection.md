# Personal Reflection

Before researching this topic, I mostly thought AI tools were simply advanced code generators that could help developers write programs faster. After researching AI coding assistants and development tools, I realized that AI is influencing almost every part of software engineering, including debugging, documentation, testing, deployment, and collaboration.

The thing that surprised me the most was how quickly AI tools have become integrated into professional software development environments. Many companies already use AI assistants daily to improve productivity and reduce repetitive work. I did not expect tools like GitHub Copilot and ChatGPT to be so widely adopted by developers around the world.

One concern I have is that some developers may become too dependent on AI-generated code. If programmers stop understanding the logic behind the code they use, software quality and security could become serious problems. AI systems can generate incorrect or vulnerable code, and inexperienced developers may not notice these mistakes.

At the same time, I see many opportunities in AI-assisted development. AI tools can help developers learn faster, solve problems more efficiently, and focus on creative and complex tasks instead of repetitive coding. These tools can also help beginners understand programming concepts more easily.

I believe that important skills in the future will include critical thinking, debugging, communication, system design, and adaptability. Even if AI becomes more advanced, companies will still need human engineers who can make decisions, understand business requirements, and verify software quality.

This research changed my opinion about AI in software engineering. I no longer see AI as a threat that will simply replace programmers. Instead, I think AI will become a powerful assistant that changes how software engineers work.

Personally, I plan to adapt by continuously learning new technologies while also improving my programming fundamentals. I want to learn how to use AI tools effectively without depending on them completely. I believe developers who combine technical skills with AI knowledge will have strong opportunities in the future software engineering industry.
